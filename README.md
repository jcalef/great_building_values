# format

The format is an array of arrays inside levels </br>
```json
{"levels": 
    [
        ["0&#8594;1", "90", "10", "5", "0", "0", "0"],
        ["1&#8594;2", "90", "10", "5", "0", "0", "0"],
        ...
    ]
}
```
</br>

Inside each array contains <br>
```
[level up range, fp to level, first place fp, second, third, fourth, fifth]
```
